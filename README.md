Blinkit dashboard

Overview
This Power BI project visualizes and analyzes Blinkit’s sales performance, item distribution, and outlet efficiency across different locations and categories.  
The dashboard provides key insights into total sales, average sales per outlet, customer ratings, and item-wise contribution, helping stakeholders understand performance trends and make data-driven decisions.

Objectives
- To analyze total and average sales across different outlet sizes and types.  
- To understand item-type performance and contribution to total revenue.  
- To track outlet establishment growth trends over time.  
- To provide insights into customer ratings and sales distribution by outlet and location.

ey Insights from the Dashboard
- Total Sales: $1.20M with an average sale of $141.  
- Total Items: 8,523 items analyzed with an average rating of 3.9.  
- Top Outlet Type: Supermarket outlets generated the highest sales ($788K).  
- Top-Selling Item Categories: Fruits, snacks, and household items are the top contributors.  
- Fat Content Analysis: Low-fat items contributed $776K, outperforming regular-fat items.  
- Growth Over Time: Sales peaked around 2018 ($205K) and showed consistent performance afterward.  
- Outlet Locations: Tier 3 cities showed the highest total sales ($472K), followed by Tier 2 and Tier 1.

Dashboard Features
- Interactive slicers for Outlet Location, Outlet Size, and Item Type.  
- Dynamic visual filters for Total Sales, Average Sales, and Average Rating.  
- Year-wise trend analysis using line charts.  
- Donut and bar charts for category-wise and outlet-wise insights.  
- Clear and user-friendly layout using Blinkit’s brand color scheme (yellow-green).

Tools & Technologies
- Power BI Desktop (Data visualization and modeling)  
- Power Query (Data transformation)  
- DAX (Calculated measures for KPIs like Avg Sales, Rating, etc.)  
- Excel / CSV Dataset (Source data)
