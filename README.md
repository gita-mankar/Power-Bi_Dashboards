Blinkit dashboard

Overview
This Power BI project visualizes and analyzes Blinkit’s sales performance, item distribution, and outlet efficiency across different locations and categories.  
The dashboard provides key insights into total sales, average sales per outlet, customer ratings, and item-wise contribution, helping stakeholders understand performance trends and make data-driven decisions.

Objectives
- To analyze total and average sales across different outlet sizes and types.  
- To understand item-type performance and contribution to total revenue.  
- To track outlet establishment growth trends over time.  
- To provide insights into customer ratings and sales distribution by outlet and location.

Key Insights from the Dashboard
- Total Sales: $1.20M with an average sale of $141.  
- Total Items: 8,523 items analyzed with an average rating of 3.9.  
- Top Outlet Type: Supermarket outlets generated the highest sales ($788K).  
- Top-Selling Item Categories: Fruits, snacks, and household items are the top contributors.  
- Fat Content Analysis: Low-fat items contributed $776K, outperforming regular-fat items.  
- Growth Over Time: Sales peaked around 2018 ($205K) and showed consistent performance afterward.  
- Outlet Locations: Tier 3 cities showed the highest total sales ($472K), followed by Tier 2 and Tier 1.

Dashboard Features
- Interactive slicers for Outlet Location, Outlet Size, and Item Type.  
- Dynamic visual filters for Total Sales, Average Sales, and Average Rating.  
- Year-wise trend analysis using line charts.  
- Donut and bar charts for category-wise and outlet-wise insights.  
- Clear and user-friendly layout using Blinkit’s brand color scheme (yellow-green).

Tools & Technologies
- Power BI Desktop (Data visualization and modeling)  
- Power Query (Data transformation)  
- DAX (Calculated measures for KPIs like Avg Sales, Rating, etc.)  
- Excel / CSV Dataset (Source data)

Customer Churn Analysis Dashboard

Overview
This Power BI project presents an interactive Customer Churn Analysis Dashboard created by analyzing the dashboard walkthrough video.
The dashboard visually explores customer churn patterns across contract types, payment methods, internet services, tenure, and demographics, enabling stakeholders
to quickly identify churn drivers and retention opportunities through intuitive visuals and KPIs.

Objectives
- To analyze customer churn behavior using interactive Power BI visuals.
- To identify key factors influencing churn, such as contract type, payment method, and internet service.
- To compare churned vs non-churned customers across different customer segments.
- To provide a clear, business-friendly dashboard that supports data-driven retention strategies.

Key Insights from the Dashboard
- Total Customers & Churn Rate: KPI cards provide an instant overview of total customers, churned customers, and churn percentage.
- Contract Impact: Month-to-month contracts show significantly higher churn compared to long-term contracts.
- Internet Service Analysis: Customers using fiber optic services exhibit higher churn levels.
- Payment Method Trends: Electronic check users have a higher churn rate than other payment methods.
- Tenure Effect: Customers with lower tenure are more likely to churn, indicating early-stage customer risk.
- Demographics: Gender and senior citizen analysis reveal variations in churn behavior across customer groups.

Dashboard Features
- KPI cards for Total Customers, Churned Customers, Churn Rate, and Average Monthly Charges.
- Clustered bar and column charts for category-wise churn comparison.
- Donut charts for service-based churn distribution.
- Interactive slicers for Contract, Internet Service, Gender, and Senior Citizen.
- Clean, modern layout with Canva-style professional color theme for better readability.

Tools & Technologies
- Power BI Desktop – Dashboard design and visualization
- Power Query – Data cleaning and transformation
- DAX – Measures for Churned Customers, Churn Rate, Avg Monthly Charges, etc.
- CSV Dataset – Telecom customer data source

Why This Project Matters
This dashboard converts complex churn data into clear visual insights, helping businesses understand why customers leave and where retention efforts should be focused.
